ZenBot API

Scrapers, descargadores y herramientas para bots de WhatsApp. Autenticación por API key en cada request.

🔑 Autenticación
Endpoints de administración
POST/auth/register
Crear nueva API key
DELETE/auth/revoke
Revocar una API key
GET/auth/keys
Listar todas las keys
Todas las rutas requieren el header x-api-key: zb_xxxx
📡 Ping
GET/pingVerificar que la API está activa — no requiere key
