🔑 ¿Cómo usar tu API Key?

Para hacer peticiones a la API desde tu código, debes incluir tu API Key en los Headers de la petición bajo el nombre x-api-key.

// Ejemplo usando Axios (Node.js / Baileys)
import axios from 'axios';

const response = await axios.get('https://api.axelmodz.xyz/scrapers/ytinfo?url=...', {
  headers: {
    'x-api-key': 'zb_TU_API_KEY_AQUI'
  }
});

console.log(response.data.result);
// Ejemplo usando Fetch (JavaScript Vanilla)
const res = await fetch('https://api.axelmodz.xyz/search/pinterest?q=gatos', {
  method: 'GET',
  headers: { 'x-api-key': 'zb_TU_API_KEY_AQUI' }
});
const data = await res.json();
